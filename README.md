# 🎓 私家家教系统 (Private Tutor System)

一个专为初三学生（中考备考阶段）设计的 AI 家教团队，通过多 Agent 协作模式，提供从每日出题、智能阅卷到学习复盘的全流程个性化辅导服务。

## ✨ 核心特性

- **📚 每日模拟出题** - 基于近5年中考真题，生成针对性练习试卷
- **📝 智能阅卷批改** - 识别学生作答，逐题评分并分析错因
- **🎯 学科专项辅导** - 针对薄弱知识点进行深度讲解
- **📊 学习数据分析** - 追踪薄弱点，生成日/周/阶段学习报告
- **📄 PDF 试卷生成** - 将试卷导出为可打印的 PDF 格式

## 📖 快速开始

### 安装

1. 下载 `private-tutor.zip`
2. 在 WorkBuddy 中安装 Skill：
   ```
   # 在 WorkBuddy 中执行
   安装 skill private-tutor.zip
   ```

### 初始化系统

首次使用时，系统会自动创建以下工作目录：

```
workspace/
├── 管家系统/              # 家教管家配置
├── 出题Agent/             # 出题Agent配置
├── 阅卷批改Agent/          # 阅卷批改配置
├── 科目专项Agent/          # 各科专项辅导配置
├── 总结复盘Agent/          # 学习复盘配置
├── 每日试卷/              # 自动存储每天生成的试卷
├── 错题本/                # 自动累积的错题记录
├── 学习报告/              # 每日/每周学习报告存档
└── PDF试卷/               # 生成的PDF试卷输出
```

### 每日工作流程

#### 第一步：早晨出题

告诉管家今天想练哪个科目：
```
"今天开始，帮我出一套语文模拟题"
"我想先做数学，重点考函数"
"今天先来一套全科综合卷"
```

#### 第二步：学生做题并上传

- 打印或手写完成试卷
- 做完后拍照发给管家

#### 第三步：智能批改与复盘

管家自动安排：
1. 阅卷批改Agent 识别并评分
2. 科目专项Agent 深度讲解错题
3. 总结复盘Agent 生成学习报告
4. 管家整合后反馈给用户

## 🛠️ 技术细节

### 包含的 Agent

- **家教管家** - 统一协调入口，90%置信度审核
- **出题Agent** - 生成每日模拟试卷
- **阅卷批改Agent** - 智能识别与评分
- **科目专项Agent** - 语文/数学/英语/物理/化学/文综深度辅导
- **总结复盘Agent** - 学习数据分析与报告生成

### 辅助脚本

- `generate_pdf.py` - Markdown 转 PDF 试卷
- `gen_figures.py` - 数学几何图形生成
- `clean_latex.py` - LaTeX 公式符号清理

## 📝 注意事项

1. **唯一对话入口**：用户只需与家教管家对话
2. **90%置信度机制**：确保输出质量
3. **试卷照片质量**：尽量拍清晰，避免模糊
4. **答案区分页**：PDF 答案单独分页，方便打印选择
5. **地区适配**：默认针对河北省石家庄市中考，可调整
